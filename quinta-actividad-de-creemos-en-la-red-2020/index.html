Quinta Actividad de Creemos en la Red 2020!

by Creemos en la Red
published on mayo 27, 2020

Llego el momento de conocer nuestro segundo lenguaje: CSS.

Este lenguaje se usa para definir el aspecto y la presentación de nuestras paginas.

Hasta ahora lo usamos indirectamente usando el CSS definido por bootstrap.

Ahora vamos a empezar a aprender como hace bootstrap para definir esas cosas que usamos.

La actividad consiste en ver el video de Introducción Selectores CSS y transcribir el HTML y CSS que se muestra. La entrega es del código que esta al final del video pero les recomendamos que prueben todos los pasos intermedios para irse acostumbrando a escribirlo, a su sintaxis e ir identificando cuales son los errores mas comunes que cometen al escribirlo.
