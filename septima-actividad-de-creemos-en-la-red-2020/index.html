Septima  Actividad de Creemos en la Red 2020!

by Creemos en la Red
published on agosto 30, 2020

Seguimos unificando conocimientos y probando como es hacer una pagina completa, esta vez con menos ayuda de nuestro querido bootstrap y siendo mejores habitantes de la web usando tags adecuados cuando la situación lo amerita.

La actividad consiste en hacer una pagina de cero, usando tags HTML semánticos y escribiendo todo el CSS "a mano", en el camino vamos a aprender a usar variables CSS para centralizar valores usados comúnmente como colores, de paso aprenderemos a hacer una version "dark" de nuestro sitio.

La actividad consiste en seguir el siguiente video, creando un sitio similar, adiciones son bienvenidas, uso de cosas ya aprendidas también, lo importante es usar los tags semánticos que se muestran, escribir todo el CSS y usar variables para crear una versión oscura del sitio.

La fecha de entrega es el lunes 14 de setiembre usando la plantilla "Actividad 7" en https://github.com/creemosenlared/entregas/issues
