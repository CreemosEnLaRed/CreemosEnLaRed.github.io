Sexta Actividad de Creemos en la Red 2020!

by Creemos en la Red
published on julio 6, 2020

Llego el momento de empezar a integrar todos los conocimientos, el resultado de esta actividad es una pagina completa con casi todas las características necesarias para hacer paginas reales, antes de empezar con eso va a ser necesario que aprendamos unos conceptos nuevos:

 

Fuentes y propiedades CSS para texto

 

Sistema de grilla de bootstrap y flexbox en CSS

 

Consigna de Actividad

La actividad consiste en ver los tres videos de este post, hacer la una pagina similar a la que se muestra en el video de abajo, el contenido puede ser de cualquier tipo siempre que:

  • Tenga un menú de navegación arriba usando row/col
  • Tenga un pie de pagina abajo usando row/col
  • Tenga al menos dos filas con al menos dos columnas cada una y que para resoluciones de pantalla chicas se muestre una sola columna
  • Use una fuente usando @import con CSS
  • Use las propiedades CSS text-shadow y transform

Se recomienda completar todos los niveles del juego https://flexboxfroggy.com/#es

A diferencia de las consignas anteriores, en esta se recomienda que copien y peguen tanto HTML y CSS como quieran, la idea es que se vayan acostumbrando a algo que van a usar muy seguido, que es elegir, copiar, pegar y adaptar código de ejemplo que encuentren en la web.

La entrega de la consigna se hace en el repositorio https://github.com/CreemosEnLaRed/entregas/issues/new/choose usando la plantilla "Entrega de Actividad 6".

La fecha limite de entrega es el 20 de Julio a las 24hs

A dejar volar la imaginación!

PD: sabemos que estamos retrazades con las correcciones, vamos a intentar ponernos al día :)
