Consigna de ingreso a Creemos en la Red 2020

by Creemos en la Red
published on febrero 22, 2020

Hola! Bienvenides a Creemos en la Red! En este curso vas a aprender los fundamentos de la programación de sitios web. Si algo no te sale, no te preocupes, las mentoras del grupo de Telegram están a disposición para responder tus consultas. Y recordá que los errores son parte del aprendizaje! Nadie nace sabiendo programar, pero todes podemos aprender.

En este primer ejercicio, vas a crear una cuenta en Github. ¿Qué es GitHub? Es una página que nos permite organizar nuestros proyectos de programación usando algo llamado Git. ¿Y qué es Git? Los programas (ya sean páginas web, software, o lo que sea que escribamos con código) no se escriben de una sola vez y ya está. En general, lleva varios días de trabajo, correcciones, reescrituras y agregados hasta que llegamos a la versión “final” (y luego tenemos que mantenerla y corregir los bugs!). Git nos ayuda manteniendo un historial de todos los cambios que hicimos a nuestro proyecto, desde la primera línea que escribimos hasta la versión final, sin importar cuán compleja sea.

También nos permite otra cosa muy importante que es colaborar con otras personas en un mismo proyecto. Si tienen cierta edad, se acuerdan cuando no existía google drive y teníamos que mandarnos archivos de word por mail para hacer un TP grupal (o juntarnos sí o sí la misma casa). Era muy incómodo! En ese sentido, Git es como un Google Drive para programadores: centraliza todos los aportes en una versión "oficial" del proyecto y así permite que varias personas trabajen en un mismo proyecto sin pisarse.


Por eso, Git (y GitHub) son usados en casi todos los trabajos de programación. Esta herramienta te va a ser intensamente útil en tu vida profesional si trabajás con código.

Para este primer ejercicio, solamente tenés que hacerte una cuenta en GitHub. Más adelante nos iremos familiarizando con cómo funciona.

El primer paso es visitar el canal de youtube de Creemos en la Red (y suscribirse si tenes ganas :)

Ver los siguientes videos:

  • Introducción a Bloques Web
  • Crear Cuenta en Github
  • Verificando la Cuenta de Github
  • Creando nuestra pagina personal en github
  • Creando nuestra pagina personal en github (resumen)

Realizar las siguientes actividades:

  • Crear su propia cuenta en github (como se muestra en el video Crear Cuenta en Github)
  • Crear una pagina usando Bloques Web (como se muestra en el video Introducción a Bloques Web)
  • Publicar la pagina como pagina personal en github  (como se muestra en los videos Creando nuestra pagina personal en github y Creando nuestra pagina personal en github (resumen))

La pagina puede ser una pagina personal, describiendo algun hobby, organizacion o tema de interes, o una propuesta para la pagina principal de Creemos en la Red.

Entregable:

  • Nombre de usuarie en Telegram
  • Nombre de usuarie en github
  • Link a pagina personal de github

Cualquier duda, consultar en el canal de telegram.
