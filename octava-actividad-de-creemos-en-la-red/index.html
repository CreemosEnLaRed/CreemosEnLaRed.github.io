Creemos en la Red

Octava Actividad de Creemos en la Red!

by Creemos en la Red
published on enero 17, 2021

Feliz año!

Estamos de vuelta con creemos en la red, en esta actividad vamos a instalar y configurar las herramientas que nos van a acompañar por mucho tiempo, ya que son las que usan las desarrolladoras para hacer paginas y aplicaciones web.

Vamos a instalar un editor de desarrollo integrado (IDE), el mas usado según las ultimas encuestas: Visual Studio Code

Luego de instalarlo vamos a instalar una extension que nos provee funcionalidad extra cuando editamos HTML y CSS

Con nuestra IDE configurada vamos a crear nuestro primer proyecto con dos archivos (index.html y style.css)

Para poder accederlos como un sitio web desde nuestro navegador vamos a necesitar un servidor web local, para eso vamos a instalar un interprete de javascript llamado deno y vamos a usarlo para instalar un pequeño programa/script que funciona como servidor web local de archivos, file_server

Vamos a iniciar este servidor de archivos "sirviendo" los archivos de nuestro proyecto y vamos a accederlo desde nuestro navegador visitando http://localhost:4507/ (localhost es un nombre que siempre apunta a la computadora en la que estamos)

Con todo nuestro ambiente de desarrollo, proyecto y servidor andando vamos a proceder a hacer una pagina que incluya elementos interactivos de otras paginas: videos, audio y mapas.

Si bien la entrega es la pagina con los elementos "embebidos" la parte mas importante de esta actividad es instalar y configurar el IDE, el plugin, el interprete y el servidor web. Por suerte estos pasos que son un poco complicados nos van a servir para el resto de nuestra carrera como desarrolladores ya que son los elementos estandar para desarrollo web.

Una vez terminada la actividad se entrega como es usual creando un issue en https://github.com/CreemosEnLaRed/entregas/issues usando la plantilla "Entrega de Actividad 8"

La entrega consiste tiene que estar en un directorio llamado actividad-8 en su sitio personal de github, similar a https://user.github.io/actividad-8 (reemplazando user con su user de github)
